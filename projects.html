Featured Project: Super App X

Super App X - Revolutionizing Smart Living

Super App X is an innovative platform designed to integrate various smart home devices and services into a single, intuitive interface. It allows users to control lighting, temperature, security, and entertainment systems seamlessly, providing a truly connected living experience. My role involved leading the backend architecture design and implementing the core communication protocols for device integration.

Key Achievements: Reduced device response latency by 30% through optimized API design; Successfully integrated over 50 different smart device models; Achieved a 99.9% uptime for the core service infrastructure under heavy load.

Technologies Used: Kubernetes Kafka Vue.js PostgreSQL gRPC AWS IoT Python
